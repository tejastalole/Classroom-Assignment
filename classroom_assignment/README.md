Project Overview:
The Classroom Assignment Management System helps manage student assignments, deadlines, submissions, and grading. It will feature student and teacher users, assignment creation, assignment submission, and basic reporting (e.g., pending, submitted, and graded assignments).

Functional Requirements:
User Management:
Teacher: Can create and grade assignments.
Student: Can submit assignments and view results.
Assignment Management:
Teachers create assignments with a title, description, deadline, and maximum score.
Students submit assignments before the deadline.
Teachers can grade assignments and provide feedback.
File Storage:
Store basic details of users and assignments in a file (.txt or .json).
Database Storage:
Store detailed information such as assignment submissions and grades using SQLite.


Features:
Teachers can create assignments.
Students can submit assignments.
Submissions are stored in an SQLite database.
User data is stored in files for persistent storage.
